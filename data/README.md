# Financial Data Sources

This folder contains datasets exported from the company's 1C:Enterprise 8.3 ERP system and used for analytical tasks during the internship at AS-Montazh LLP.

## Included Datasets

### account_analysis.xlsx

Analysis of accounting account balances and turnovers used for cash position assessment and financial monitoring.

### inventory_movement.xlsx

Inventory transactions and stock movement data used for inventory turnover and warehouse efficiency analysis.

### inventory_turnover.xlsx

Dataset used to evaluate inventory turnover rates and identify slow-moving items.

### inventory_register.xlsx

Material and inventory records containing product-level information for inventory structure analysis.

### product_catalog.xlsx

Product nomenclature dataset containing SKU descriptions and classification information.

### working_capital.xlsx

Dataset used to analyze the structure and dynamics of working capital components.

### cash_inflow.xlsx

Cash receipts data used for cash flow and liquidity analysis.

### cash_outflow.xlsx

Cash payments data used for expenditure structure and cash utilization analysis.

### accounts_receivable_dynamics.xlsx

Customer debt data used to analyze receivables concentration, payment behavior, and collection efficiency.

### accounts_payable_dynamics.xlsx

Supplier debt data used to evaluate payable structure and supplier concentration risks.

### profit_and_loss.xlsx

Revenue and expense data used for profitability and financial performance analysis.

### material_statement.xlsx

Material inventory records used for stock valuation and inventory composition assessment.

## Purpose

The datasets were used for:

* Inventory Analysis
* Working Capital Analysis
* Accounts Receivable Analysis
* Accounts Payable Analysis
* Cash Flow Analysis
* Financial Performance Evaluation
* KPI Monitoring and Business Reporting

All analyses were performed using Microsoft Excel, Python (Pandas, NumPy, Matplotlib), Google Colab, and data exported from 1C:Enterprise 8.3 ERP.
