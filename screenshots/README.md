# 1C ERP Screenshots

## Included Reports

- Account 1030 Analysis (Cash Account)
- Profit and Loss Analysis

These reports were used to analyze the company's financial position, liquidity, cash flow, customer debt, supplier obligations, and operational performance during the internship period.
